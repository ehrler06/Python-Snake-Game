Snake Game - Python (Turtle Module)
Overview
This is a simple Snake game implemented in Python using the turtle module. The player controls a snake, trying to eat food to grow in length. The game ends when the snake collides with itself or the boundaries of the screen.

Features
Smooth and responsive snake movement.
Simple score tracking.
Randomly placed food items.
Collision detection with boundaries and the snake's own body.
Controls
W - Move Up
S - Move Down
A - Move Left
D - Move Right
How to Run
Install Python: Ensure Python is installed on your system. You can download it from python.org.

Open Visual Studio Code:

Launch Visual Studio Code.
Create a New File:

Open a new file (File > New File).
Paste the Script:

Copy the Python script from this project and paste it into the new file.
Save the File:

Save the file with a .py extension, e.g., snake_game.py.
Run the Script:

Run the script by pressing Ctrl + F5 or selecting Run > Run Without Debugging from the menu.
Game Rules
The snake moves in the direction indicated by the player.
Eating food increases the snake's length and the score.
The game ends if the snake collides with the boundaries of the screen or its own body.
